Python模板注入 SSTI学习

0x01:SSTI是什么？

​ 首先，针对一些就算没有学过Python的朋友，SSTI还是很友好的。

​ SSTI（Server-Side Template Injection）服务器端模板注入，也有人称它为python沙盒绕过；举个不恰当的例子，高中的时候抄作业，要抄的答案就可以当作是一个模板，而此时你抄的内容就是模板的内容，每个人的写入题目的模板可以不同，但是答案（模板）是多变的，你可以拿各种别人写好的模板填在你的题目里；但是，在不当使用模板进行渲染时，就会产生模板注入。
